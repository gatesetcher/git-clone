# Проект: Проектирование и изготовление исторических рам с использованием 3D-технологий

**Участники проекта:** 18 человек

---

## 1. Исследовательский этап и выбор технологии
В начале работы над проектом наша группа провела серию выездных мероприятий. Мы посетили крупнейшие **музеи и художественные выставки**, где детально изучали старинные рамы для картин. Основное внимание уделялось:
* Сложности резьбы и декоративных элементов;
* Сохранности исторических материалов;
* Геометрии и симметрии конструкций.

В ходе обсуждения различных методов реставрации и воспроизведения (от ручной резьбы до фрезеровки на станках ЧПУ), группа приняла решение остановиться на связке современных технологий: **3D-сканирование** для получения точной цифровой копии и **3D-печать** для создания итогового изделия.

---

## 2. Сравнение материалов для 3D-печати
Для того чтобы рама выглядела аутентично, мы провели сравнительный анализ обычного пластика и специализированного композита.


| Характеристика | Обычный пластик (PLA) | Пластик с древесным наполнителем (Wood PLA) |
| :--- | :--- | :--- |
| **Состав** | Полилактид (биопластик) | PLA + ~15-30% измельченных волокон дерева |
| **Внешний вид** | Гладкий, глянцевый, "пластиковый" | Матовый, текстурный, имитирует дерево |
| **Запах при печати** | Сладковатый (кукурузный) | Запах жженой древесины |
| **Постобработка** | Сложно шлифуется, требует грунтовки | Легко шлифуется, режется, покрывается морилкой |
| **Тактильные ощущения** | Холодный пластик | Теплый, напоминает натуральный материал |

## 3. Техническое оснащение: 3D-сканирование
Для создания высокоточной цифровой модели исторической рамы мы используем промышленный ручной 3D-сканер **Shining 3D Einscan HX2**. Этот прибор позволяет работать с объектами разного размера и сложности.

### Основные характеристики сканера:
* **Гибридный источник света:** Сочетание лазера (для темных или блестящих поверхностей) и светодиодного (LED) сканирования.
* **Высокая точность:** До 0.04 мм в режиме лазерного сканирования.
* **Скорость сканирования:** До 1,200,000 точек/сек, что позволяет быстро оцифровать габаритную раму.
* **Встроенная камера:** Позволяет захватывать текстуру и цвет объекта.
* **Адаптивность:** Отлично справляется со сложными мелкими узорами и глубоким рельефом старинной резьбы.

## 4. Дальнейшие планы
На текущий момент проект находится на стадии активной реализации. В планах проектной деятельности на будущее остается:
* Завершение финальной печати всех сегментов рамы;
* **Постобработка напечатанной рамы:** шлифовка, склейка, художественная покраска и патинирование для достижения эффекта состаривания.
